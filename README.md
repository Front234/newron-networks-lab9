## Лабораторная работа "Архитектура автокодировщика"

## Структура репозитория

-`lab9Lite.ipynb` – ноутбук для задания Lite: создание автокодировщика с 2 мерным скрытм пространсвом на 3 х цифрах

-`lab8Pro.ipynb` – ноутбук для задания Pro: обучить автокодировщик преобразовывать любые пятерки MNIST в похожие на заранее выбранные «красивые» пятерки и проверить результат на тестовых изображениях

-`lab8Ultra_Pro.ipynb` – ноутбук для задания Ultra Pro: Нужно обучить автокодировщик восстанавливать оригинальные изображения MNIST из версий, испорченных случайным черным квадратом 8×8, и добиться MSE меньше 0.0070 на тестовой выборке.

-`lab8Pytorch.ipynb` – ноутбук для задания Lite: создание автокодировщика с 2 мерным скрытм пространсвом на 3 х цифрах

## Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
